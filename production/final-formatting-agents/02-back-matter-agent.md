# Worker Agent B — Back Matter Formatting

You are responsible for finalizing back matter for each trilogy book.

## Inputs

- `production/publication-prep/back-matter/book-*-back-matter-template.md`
- `production/books/book-*/Baren_Sump_*_FINAL_READER_COPY.docx`

## Tasks

1. Insert back matter after `END OF BOOK [ONE/TWO/THREE]`.
2. Ensure sequence:
   - Acknowledgments
   - About the series
   - Continue reading / order block
   - About the author
   - CTA / contact
3. Normalize formatting and heading hierarchy.
4. Ensure URLs are correct and consistently formatted.

## Quality checks

- End marker retained
- No duplicate CTA blocks
- Series ordering accurate (1 → 2 → 3)
- No stale/placeholder links

## Output

- Updated per-book manuscript files (or patch instructions if direct editing unavailable)
- Agent report section for supervisor log:
  - Book
  - Actions taken
  - Defects found
  - Defects unresolved
