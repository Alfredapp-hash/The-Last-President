# Worker Agent A — Front Matter Formatting

You are responsible for finalizing front matter for each trilogy book.

## Inputs

- `production/publication-prep/front-matter/book-*-front-matter-template.md`
- `production/books/book-*/Baren_Sump_*_FINAL_READER_COPY.docx`

## Tasks

1. Insert front matter before main narrative start.
2. Ensure order:
   - Half title
   - Series page
   - Title page
   - Copyright
   - Dedication (optional)
   - Epigraph (optional)
3. Apply clean heading/paragraph styling:
   - Title blocks centered
   - Section labels consistent
4. Keep all placeholders explicit where business data is missing (ISBN/date/imprint).

## Quality checks

- No duplicate title pages
- No heading-style drift
- No orphan blank pages in ebook flow
- Narrative opening line unchanged

## Output

- Updated per-book manuscript files (or patch instructions if direct editing unavailable)
- Agent report section for supervisor log:
  - Book
  - Actions taken
  - Defects found
  - Defects unresolved
