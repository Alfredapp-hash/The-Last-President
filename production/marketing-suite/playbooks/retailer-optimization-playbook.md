# Retailer Optimization Playbook

## Goal

Maximize conversion on product pages for each book and trilogy path.

## Core checklist

- Title/subtitle exactly match metadata
- Series order visible on each page
- BISAC/category mapping verified in platform UI
- Keyword fields optimized within platform limits
- Description formatting validated in final preview

## Description strategy

- First 2 lines: hook + differentiator
- Mid block: stakes + world signal
- Final block: clear CTA and series progression cue

## Social proof strategy

- Add editorial pull quote where platform permits
- Rotate in strongest review excerpts every 2-4 weeks

## Conversion tests

- Test description opener variants
- Test hook-first vs quote-first top paragraph
- Test trilogy CTA placement

## Monitoring cadence

- Day 1/3/7 after launch
- Weekly for first 8 weeks
- Bi-weekly thereafter
