# Organic Social Playbook

## Platforms

- X / Threads
- Instagram
- TikTok / Reels / Shorts
- LinkedIn (select thought-leadership angles)

## Content pillars

1. Hook lines (high-concept entry)
2. Pull quotes (text-over-image)
3. Case-file artifacts (visual lore)
4. Author POV on themes (institutions, language, accountability)
5. Reader reactions / social proof

## Posting frequency

- Short-form text: daily
- Visual post: 3-5 per week
- Video snippets: 2-4 per week
- Story/poll: 3 per week

## Format templates

### Template A — Hook

1. One provocative line
2. One sentence context
3. CTA: "Start with Book One"

### Template B — Quote card

1. Quote
2. Book + chapter attribution
3. CTA link

### Template C — Mini-thread

1. Core thesis statement
2. 3-point unpack
3. Trilogy CTA

## Engagement protocol

- Reply to meaningful comments within 12h
- Pin best reader reactions weekly
- Save top objections for future clarifying posts

## Weekly targets

- Saves/shares up
- Click-through rate stable or rising
- Profile-to-site conversions up week over week
