# Baren Sump Trilogy — Full Marketing Suite

Execution-ready marketing system for launching and sustaining the trilogy:

- **Book One:** The Last President
- **Book Two:** Children of Tomorrow
- **Book Three:** The Black Path

This suite extends the baseline copy in `production/marketing/` into a full campaign operating system.

## Suggested run order

1. `01-go-to-market-strategy.md`
2. `02-brand-messaging-house.md`
3. `03-launch-timeline-90-day.md`
4. `playbooks/*.md` (channel execution)
5. `copy/*.md` (campaign copy banks)
6. `calendars/content-calendar-8-weeks.csv`
7. `ops/campaign-tracker.csv` and `ops/weekly-growth-dashboard-template.md`
8. `assets/creative-briefs.md`

## Folder map

- `playbooks/` — channel-by-channel execution instructions
- `copy/` — reusable ad, social, video, and CTA copy
- `calendars/` — publication calendar and posting schedule
- `ops/` — KPI tracking, cadence, launch command center
- `assets/` — asset specs and creative briefs

## Core KPIs

- Email signups/week
- Pre-orders / direct sales
- CTR by channel
- Landing page conversion rate
- ARC/review volume and sentiment
- Cost per lead (paid)
- Cost per purchase (paid)
