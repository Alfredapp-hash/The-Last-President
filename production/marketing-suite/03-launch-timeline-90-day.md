# 03 — 90-Day Launch Timeline

## Phase 1: Foundation (Days 1-14)

- Finalize metadata checks per platform taxonomy
- QA site conversion path (homepage -> press -> retailer links)
- Build email capture flow and welcome automation
- Prepare outreach lists (reviewers, podcasts, newsletters, BookTok/Bookstagram)
- Lock asset pack versions and naming

## Phase 2: Audience Build (Days 15-35)

- Publish 3-5 organic posts/week (hook + quote + visual artifact rotation)
- Send weekly newsletter ("Ledger Dispatch")
- Start ARC reviewer outreach
- Secure first wave of interviews/features
- Run low-budget paid tests (awareness + list growth)

## Phase 3: Conversion Push (Days 36-60)

- Switch CTA focus to purchase/pre-order
- Deploy retailer-optimized blurbs and A/B variants
- Run retargeting and engaged-audience campaigns
- Trigger launch-week email sequence
- Coordinate social countdown across all channels

## Phase 4: Launch + Sustain (Days 61-90)

- Execute launch-day command center runbook
- Track daily KPI panel
- Push review-acquisition campaigns
- Promote trilogy ladder path (Book 1 -> Book 2 -> Book 3)
- Publish post-launch media hits and reader social proof

## Weekly cadence

- Monday: KPI review + sprint planning
- Tuesday: Long-form content + outreach sends
- Wednesday: Paid optimization + creative refresh
- Thursday: Press/community follow-up
- Friday: Reporting + next-week prep

## Critical launch-day checklist

- Site links verified
- Retail pages live
- Email blast sent
- Social posts scheduled and published
- Paid campaigns enabled
- Customer support inbox monitored
