# Creative Briefs — Marketing Suite

## Creative system

Visual style remains "haunted evidence":

- graphite + cream base
- book accent colors
- no glossy fantasy look
- no generated baked-in text for final marketing assets

## Asset list by priority

### Tier 1 (must-have)

1. Series hero static (1:1, 4:5, 16:9)
2. Book One conversion ad set (3 images + 2 short videos)
3. Quote cards (minimum 12)
4. Launch-day story set (minimum 6 frames)

### Tier 2 (scale)

1. Character/world artifact cards
2. Behind-the-ledger author explainers
3. Review/social proof template cards

## Dimensions

- Feed square: 1080x1080
- Feed portrait: 1080x1350
- Story/reel: 1080x1920
- X image: 1600x900
- Email hero: 1200x600

## Copy pairing rules

- Lead with one clean hook line
- Keep overlay text short (< 12 words)
- Use one CTA per creative
- Always include source/attribution for quote creatives

## QA before publish

- Spelling/casing check against metadata sheet
- Contrast/readability on mobile
- CTA link and UTM parameters verified
