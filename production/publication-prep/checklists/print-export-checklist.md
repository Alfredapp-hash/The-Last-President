# Print Export Checklist (Paperback / Hardcover)

Run this per book after EPUB source is finalized.

## 1) Interior setup

- [ ] Select trim size (common: 5.5x8.5, 6x9, or 5x8)
- [ ] Set margins and gutter based on page count
- [ ] Set consistent running headers/footers (if used)
- [ ] Confirm chapter starts begin on correct page side (if required)
- [ ] Confirm widow/orphan controls meet house preference

## 2) Typography and readability

- [ ] Body font size and line spacing tested on print proof
- [ ] Paragraph indents and scene breaks are consistent
- [ ] Special glyphs/quotes/dashes render correctly
- [ ] No clipped italics or kerning collisions

## 3) Front/back matter print checks

- [ ] Copyright page includes print ISBN
- [ ] Series order page included and accurate
- [ ] About the author + CTA included
- [ ] Any legal notices required by imprint included

## 4) PDF export

- [ ] Export as PDF/X-1a or platform-recommended print profile
- [ ] Embed all fonts
- [ ] Include bleed only if interior has bleed elements
- [ ] Confirm grayscale/rich black settings for interior art (if any)

## 5) Cover wrap prep (outside manuscript scope)

- [ ] Build full wrap from final page count + paper type + trim
- [ ] Place barcode in retailer-safe zone
- [ ] Confirm spine width from final interior PDF page count
- [ ] Export print cover as press-ready PDF

## 6) Print proof validation

- [ ] Order or preview print proof
- [ ] Check chapter transitions, page breaks, and title pages
- [ ] Check footer/header consistency
- [ ] Sign off no text truncation/crop issues

## 7) Final naming convention

- [ ] `baren-sump-book-1-interior-print_v1.pdf`
- [ ] `baren-sump-book-2-interior-print_v1.pdf`
- [ ] `baren-sump-book-3-interior-print_v1.pdf`

Record final filenames in `distribution-submission-checklist.md`.
